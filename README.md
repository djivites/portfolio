# Jivites AI Portfolio 🌿

A futuristic, highly interactive personal portfolio built using **Next.js**, **Tailwind CSS**, and **Framer Motion**. It features interactive canvas elements, custom-themed glassmorphism grids, and particle-based responsive menus.

## ✨ Features

- **🌐 Holographic Portal HUD**: The landing page functions as an offline system. Connect the power cable to the chest reactor to trigger particle blasts and boot the portfolio.
- **🏠 Cybernetic Home Control**: Seamless navigation dashboard displaying personal branding, links to social nodes, and glowing curved interactive menu items.
- **🌿 Interactive Projects Catalog**: Accessible at `/projects` with custom leaf motifs and a sunlit forest background. Features 4 project cards with dedicated theme glows:
  - **MaatramMark** (Megaphone Icon - Green Glow)
  - **SkillBridge** (Network Node Icon - Cyan Glow)
  - **VentureMatch** (Handshake Icon - Amber Glow)
  - **AI News Credibility Index** (Newspaper Icon - Purple Glow)
- **👤 Sci-Fi About Page**: Showcases traits, quick academic stats, locations, and personal mission in glassmorphic widgets.

## 🛠️ Tech Stack

- **Core**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Production Build

To build the application for production:

```bash
npm run build
```

This compiles optimized code and outputs static pages for blazing-fast loading speeds.
